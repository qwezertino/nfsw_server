# Система запчастей в NFSW

## Коротко

Каждая запчасть имеет **два параметра**: грейд (★ звёздочки) и цвет (качество внутри грейда).

---

## Грейды (★ звёздочки) — мощность детали

| Грейд | ★ | В магазине? | Только из паков? |
|-------|---|-------------|-----------------|
| **Street** | ★1 | ✅ Да | — |
| **Race** | ★2 | ✅ Да | — |
| **Pro** | ★3 | ❌ Нет (disabled) | ✅ Только из Gold Pack |
| **Uber** | ★4 | ❌ Нет (disabled) | ✅ Только из Platinum Pack |
| **Elite** | ★5 | ❌ Нет (disabled) | ✅ Только из Black Diamond Pack |

> Pro/Uber/Elite существуют в БД, но отключены в магазине.
> Единственный способ получить их — через карточные паки.
> Это и делает паки ценными, а не просто скидкой.

---

## Цвета — качество внутри грейда

Внутри каждого грейда детали делятся на 5 цветов по качеству:

| Цвет | Качество | Примерный множитель цены |
|------|----------|--------------------------|
| **Grey** (серый) | Базовое | ×1.0 |
| **Common** (белый) | Обычное | ×1.4 |
| **Uncommon** (зелёный) | Хорошее | ×1.8 |
| **Rare** (синий) | Редкое | ×2.3 |
| **Elite** (золотой) | Топ | ×3.0 |

## Как это выглядит в паках

Reward table типа `street_improved_parts` даёт **случайный тип** запчасти (engine/transmission/etc) одного стиля Street. Внутри стиля могут выпасть все цвета — grey, common, uncommon, rare, elite.

| Пак | Грейд | Цвета которые могут выпасть |
|-----|-------|-----------------------------|
| Bronze Pack | Street ★1 | Grey → Common → Uncommon → Rare → Elite |
| Silver Pack | Race ★2 | Grey → Common → Uncommon → Rare → Elite |
| Gold Pack | Pro ★3 | Grey → Common → Uncommon → Rare → Elite |
| Platinum Pack | Uber ★4 | Grey → Common → Uncommon → Rare → Elite |
| Black Diamond | Elite ★5 | Grey → Common → Uncommon (Rare/Elite отсутствуют в игре) |

---

## Итог

- **Грейд** = сила детали (прямо влияет на характеристики машины)
- **Цвет** = насколько "прокачанная" деталь своего грейда
- **Pro/Uber/Elite** недоступны в магазине → паки единственный способ их получить
- Паки дают **случайный цвет** внутри грейда — можно выбить и Grey, и Elite одного грейда
